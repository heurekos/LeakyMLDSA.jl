# LeakyMLDSA.jl

[![License](https://img.shields.io/badge/license-MIT-brightgreen.svg?style=flat)](LICENSE)
[![Documentation (dev)](https://img.shields.io/badge/docs-dev-blue.svg)](https://heurekos.github.io/LeakyMLDSA.jl/dev)

This repository contains the Julia implementation of our NLP attack described in [Section 6](https://eprint.iacr.org/2026/580.pdf#section.6) of the [IACR ePrint 2026/580](https://eprint.iacr.org/2026/580). Examples demonstrating how to use the package can be found in the [documentation](https://heurekos.github.io/LeakyMLDSA.jl).
